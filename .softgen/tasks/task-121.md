## Notes
Expose the recycle bin inside the Site Personnel area and make it easy to open from the existing tab layout. The recycle bin should clearly show deleted records, their origin tab, key identifying details, deletion time, and available actions.

## Checklist
- [ ] Add a recycle bin entry point inside the Site Personnel module navigation or tab strip
- [ ] Build a recycle bin view with grouped or filterable deleted items from linked Site Personnel tabs
- [ ] Show item details that help users identify the deleted record, including source tab and relevant metadata
- [ ] Add visible restore and permanent delete actions with clear affordances and confirmation for destructive deletion

## Acceptance
Users can open the recycle bin directly from the Site Personnel module.

Each deleted item shows where it came from and what it represents.

Restore and permanent delete actions are available from the recycle bin UI.
